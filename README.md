### 特性
* 语法高亮基于[prism](https://prismjs.com/)
* Bootstrap4 flex布局

### 使用
```
cd sitedir
git clone https://github.com/wuzehv/hugo-freedom.git themes/hugo-freedom
cp themes/hugo-freedom/config.example.toml ../../config.toml
```
